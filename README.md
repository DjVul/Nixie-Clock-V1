This repository contains the hardware design and firmware resources for a 6-digit IN-12 Nixie tube clock. The device is powered via USB Type-C and uses an flyback converter to generate the high voltage required to drive the Nixie tubes.

The clock features an OLED display used for a graphical user interface, allowing the user to navigate settings and view additional information. An onboard buzzer provides audible feedback, alarms, and notification signals.

The goal of this repository is to provide all the files required to reproduce, manufacture, and program the clock. It is intended for electronics enthusiasts, hobbyists, and developers interested in Nixie tube technology, high-voltage power design, and custom hardware projects.

## Features

- **6-digit IN-12 Nixie tube display**
- **Low-profile design**
- **USB-C powered**
- **Integrated flyback converter** for Nixie high-voltage generation
- **OLED display** for graphical user interface
- **Onboard buzzer** for alarms and notifications
- **Microcontroller-based control logic**

## Repository Contents

- **Altium Designer project** – complete PCB design files
- **Schematics** – circuit diagrams of the system
- **Gerber files** – ready for PCB manufacturing
- **Firmware binary** – compiled file for programming the microcontroller
